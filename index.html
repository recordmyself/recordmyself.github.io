ug安装破解

UG是Unigraphics的缩写，是集计算机辅助设计、制造、分析与一体的软件集成系统

安装并破解UG NX11.0

1、下载好压缩文件，解压
2、以管理员身份运行UGNX11-JAVA-64位虚拟机
3、一直下一步到关闭为止

4、以管理员身份运行Launch，并点击Install License
5、选择中文简体，一直下一步知道提示选择许可文件，
6、打开UG11.064bit\破解文件\PLMLicenseServer下的splm8.lic用记事本打开，把this-host换成计算机名称（计算机属性中可以查到）,保存
7、继续安装选择修改后的文件，一直下一步直到完成

8、回到主界面点击Install NX,选择中文简体，一直下一步知道完成（语言、路径可修改，许可文件不可改）
9、复制UG11.064bit\破解文件下的两个文件夹NX11.0和Pimslicenseserver，替换安装路径下的对应文件夹
10、在开始菜单中选择lmtools以管理员身份运行
11、选中start/stop/reread选项卡，勾选Force server shotdowm 再点击stop server，取消Force server shotdowm，最后点击start server(注意先后顺序)
12、当显示server start seccessful后点击关闭
