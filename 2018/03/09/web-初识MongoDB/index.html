初识MongoDB

  • web
  • 数据库

安装MongoDB

###下载并安装MongoDB软件和以管理员身份打开命令行
1、windows->附件->cdm-右键以管理员身份运行
2、C:\windows\system32\cmd-右键以管理员身份运行

配置：

1、在软件同级目录下创建data/db和data/log文件夹，并在log文件夹下创建config.log

启动MongoDB

1、以管理员身份打开命令行
2、进入MongoDB安装文件夹下的bin文件夹
3、执行{path}/mongod.exe –dbpath {path}/db

4、{path}mongo.exe 链接MongoDB

环境变量配置

1、Path的配置：path可以看成一个全局变量，当要打开一个程序时，会查找到程序的路径，配置环境变量后，就可以直接使用程序文件。也就是说当将一个应用程序配置了环境变量时，他的路径就可以在全局使用了。环境变量就是一个寻找路径的关系
