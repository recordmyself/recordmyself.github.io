2、位运算符
& 按位与：对应为都为1，结果为1，否则为0（判断奇偶数）
| 按位或：对应位都为0，结果为0，否则为1（取整 3.1 | 0 //3）
^ 按位异或：对应位不同，结果为1，否则为零（同或等于异或取反或者对应位相同，结果为1，否则为0）
~ 按位取反：0变1,1变0（取整）
<<按位左移（2的n次方 2 << 3 //8）;>>按位右移（取半50 >> 1 //25）
（23 //8）同按位左移，但2位不能为负数

正数的原码 = 反码 = 补码
负数的反码 = ~原码; 补码 = 反码 + 1
*注：计算机中存储的都是补码
